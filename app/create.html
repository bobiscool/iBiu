进一步配置 打开目录
iBiu已将所有配置文件保存在指定的目录，按照下面的流程可以启动项目。 请预先全局安装 Node.js、npm 、webpack。

安装依赖

$ npm install

启动服务（开发环境）

$ npm run dev

服务启动后，打开 http://localhost:8080 访问网站

编译打包（生产环境）

$ npm run build
